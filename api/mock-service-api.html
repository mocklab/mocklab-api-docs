post /__admin/mappings/new

Create a new stub mapping

  • Request
  • Response

Body

Type: application/json

Example:

{
    "request": {
        "method": "GET",
        "url": "/some/thing"
    },
    "response": {
        "status": 200,
        "body": "Hello world!",
        "headers": {
            "Content-Type": "text/plain"
        }
    }
}

HTTP status code 201

Successfully created
